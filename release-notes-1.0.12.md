# Luna Log v1.0.12

## UI / Interaction

- Replace the record form modal with an interruptible bottom sheet that supports velocity handoff, snap, and rubber-banding.
- Add press-scale feedback, translucent navigation chrome, and refined typography hierarchy.
- Converge home filters into a compact horizontal range control and layered detail fields in the form.

## Motion

- Calendar supports horizontal swipe month changes and selection scale feedback.
- Insights bars and timeline cards use spring / staggered enter animations.
- Respect reduced-motion and reduced-transparency system preferences.

## Feedback

- Add meaningful haptics for save, delete, sheet snap, date selection, and period actions.
- Keep update checks on the official GitHub Release page only.
