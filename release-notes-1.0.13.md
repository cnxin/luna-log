# Luna Log v1.0.13

## Content-first recording

- Focus the home screen on current context and adding a record; move filtering and low-frequency actions behind on-demand controls.
- Use compact list rows for continuous records, with separate edit and more-actions touch targets.
- Restore unfinished drafts for new records and keep failed saves in place with an immediate retry path.

## Calendar, theme, and accessibility

- Show readable multi-event calendar markers and text summaries for insights, imports, and exports.
- Add shared theme tokens and follow the system dark appearance.
- Use 44pt high-frequency touch targets and correct single-choice versus multi-choice accessibility roles.
