## 测试步骤
1. 运行 `npx expo start`。
2. 使用 Expo Go 扫码打开应用。
3. 检查 18 个 SVG 图标是否正常显示。
4. 点击时长数值框，确认滚轮定位到当前值。
5. 使用计时器完成一次开始、暂停、完成流程，确认回填分钟数。
6. 添加记录后关闭并重开 App，确认本地数据保留。
7. 在设置页导出数据，再导入同一份 JSON，确认数据恢复。
8. 在关于页检查更新，若存在新版本，测试内置下载入口。
